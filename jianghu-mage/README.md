# 江湖魔法师 · 联机版 v2

3-6 人 · 单房间 · 多局赛制 · 实时同步

## ✨ v2 新增

- 🔮 **奥秘牌堆**：开局抽 4 张暗牌作为推理依据，4 号"窥视"私下查看其中一张
- 🎲 **3 号治愈术调整**：回血范围 1-3（原 1-6）
- ⏰ **30 秒倒计时**：每个决策超时自动扣 1 血并结束回合
- 🏆 **多局赛制**：开局选 1/3/5/7 局，每局赢家 +1 分，跑完总局数宣布冠军
- ✏️ **自定义名字**：大厅里直接编辑自己的座位名（最多 12 字）
- 🚪 **投票退出**：游戏中如果卡住，2 人投票即可中止本局返回大厅

## 🚀 部署到 Render（推荐）

1. 把整个 `jianghu-mage/` 推到 GitHub
2. https://render.com → New + → Blueprint → 选你的仓库
3. 自动读取 `render.yaml` 创建服务，1-2 分钟构建完成
4. 拿到网址 `https://xxx.onrender.com` 发给朋友

> 免费版 15 分钟无访问会休眠，下次访问需 30-60 秒冷启动。

## 🧪 本地运行

```bash
cd server
pip install -r requirements.txt
uvicorn main:app --reload
```

http://localhost:8000 ，多开几个浏览器窗口测试 3-6 人。

## 🎮 玩法摘要

1. 第 1 个人进入会自动占座位 1，复制邀请链接发给朋友
2. 大厅里可以改名字、选总局数（1-7 局）
3. ≥3 人开始；每局结束自动算分，跑完总局数后宣布冠军
4. 游戏中点"投票返回大厅"，2 人同意即中止本局（分数不变）

## 📜 设计要点

- **服务器权威**：所有规则判定（喊牌对错、扣血、超时）都在后端，前端只发指令
- **视角隔离**：每个玩家只收到自己视角的状态（看不到自己手牌、看到别人手牌）
- **奥秘堆**：mystery_revealed 是公开信息，但 mystery_deck 内容只在揭示者的 peeked_mysteries 中可见
- **断线重连**：刷新页面后会用座位号自动重连，保留游戏状态和分数
- **跨局保留**：玩家分数、姓名跨局保留，整轮结束才清零

## 🐛 已知限制

- 单房间，所有人共用同一个房间
- 没做反作弊（朋友局够用，理论上能伪造 WebSocket 消息控制自己的座位）
- 服务器重启会丢失当前游戏状态
